###基本语法###
1、语句和表达式
 语句：为了进行某种操作，一般不需要返回值
 表达式：为了得到返回值，一定有返回值
 
 2、变量
 	（1）用var重新声明一个已经存在的变量是无效的
	```
	var x = 1;
	var x;
	x   //1
	```

	（2）重复声明的变量会覆盖

	```
	var x = 1;
	var x=2;
	x  //2
	```

3、变量提升
JavaScript 引擎的工作方式：：先解析代码，获取所有被声明的变量，然后再一行一行地运行。这造成的结果，就是所有的变量的声明语句，都会被提升到代码的头部，这就叫做变量提升（hoisting）
```
console.log(a);//undefined
var a = 1;
```

6、对于var命令来说，JavaScript 的区块不构成单独的作用域（没有块级作用域）
